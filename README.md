## Resumo do Sistema
O sistema desenvolvido é um Quiz em C# utilizando Windows Forms no SharpDevelop, que permite ao usuário visualizar uma tela de abertura (Splash) ao iniciar o programa. Após a abertura, o usuário pode realizar login utilizando nome e e-mail, ou realizar o cadastro informando nome, e-mail, CPF e telefone.

Os dados dos usuários são armazenados em arquivos de texto locais. Após o login, o usuário é direcionado para a tela principal, onde pode acessar o jogo de perguntas e respostas. No Quiz, o usuário responde questões de múltipla escolha e recebe pontos conforme seus acertos.

Ao final, o sistema permite a visualização de um placar com o ranking das pontuações salvas em um arquivo de texto.

## Manual de Instruções
1. Abra o programa e aguarde o carregamento da tela Splash.
2. Será exibida a tela de Login, onde o usuário deve informar seu nome e e-mail.
3. Caso ainda não possua cadastro, acesse a tela de Cadastro e preencha:
   - Nome  
   - E-mail  
   - CPF  
   - Telefone  
4. Após o login bem-sucedido, o usuário terá acesso à tela principal.
5. Inicie o Quiz clicando na opção de jogo.
6. Leia cada pergunta e selecione a alternativa correta.
7. A cada resposta correta, o sistema soma pontos automaticamente.
8. Ao final das questões, acesse o Placar para visualizar a classificação dos jogadores.
