Tecnica de programação em algoritmo
O sistema desenvolvido é um Quiz em C# utilizando Windows Forms no SharpDevelop, que permite ao usuário visualizar uma tela de abertura (Splash) ao iniciar o programa. Após a abertura, o usuário pode realizar login utilizando nome e e-mail, ou realizar o cadastro informando nome, e-mail, CPF e telefone. Os dados dos usuários são armazenados em arquivos de texto locais. Após o login, o usuário é direcionado para a tela principal, onde pode acessar o jogo de perguntas e respostas. No Quiz, o usuário responde questões de múltipla escolha e recebe pontos conforme seus acertos. Ao final, o sistema permite a visualização de um placar com o ranking das pontuações salvas em um arquivo de texto.
Para utilizar o sistema, o usuário deve abrir o programa e aguardar o carregamento da tela Splash. Em seguida, será exibida a tela de Login, onde o usuário deve informar seu nome e e-mail. Caso ainda não possua cadastro, é necessário acessar a tela de Cadastro e preencher os dados solicitados, como nome, e-mail, CPF e telefone. Após o login bem-sucedido, o usuário terá acesso à tela principal, onde poderá iniciar o Quiz. Durante o jogo, basta clicar na questão desejada, ler a pergunta e selecionar a alternativa correta. A cada resposta correta, o sistema soma pontos automaticamente. Ao finalizar todas as questões, o usuário poderá visualizar o placar, que exibe a classificação dos jogadores de acordo com a pontuação armazenada no arquivo do sistema.
